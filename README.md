# MVP - Machine Learning

Este repositório contém o notebook e o dataset utilizados no desenvolvimento do MVP da disciplina de Machine Learning.

## Arquivos

- `MVP_Machine_Learning.ipynb` — Notebook com todas as etapas do projeto, incluindo análise exploratória, pré-processamento, modelagem, otimização e avaliação dos modelos.
- `online_shoppers_intention.csv` — Dataset utilizado no projeto.

## Dataset

O conjunto de dados utilizado foi o **Online Shoppers Purchasing Intention Dataset**, disponível publicamente no UCI Machine Learning Repository.

Objetivo do projeto: prever se uma sessão de navegação resultará em uma compra (`Revenue`).

## Execução

O notebook pode ser executado diretamente no Google Colab.

O dataset é carregado automaticamente a partir deste repositório utilizando a URL pública do GitHub, não sendo necessário realizar upload manual de arquivos.

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab
